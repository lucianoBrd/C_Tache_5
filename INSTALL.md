# Tache 5

### Compiler le projet
```
cd path_project/src
make
```
### Executer les tests
```
cd path_project/src
make tests
```
### Executer le projet
Pour executer le/les client(s) :
```
cd path_project/src
./client path_image
```
path_image est optionnel.

Pour executer le serveur :
```
cd path_project/src
./serveur
```
### Supprimer les fichiers compil√©s
```
cd path_project/src
make clean
```

# Tache 4

### Compiler le projet
```
cd path_project/src
make
```
### Executer le projet
Pour executer le client :
```
cd path_project/src
./client path_image
```
path_image est optionnel.

Pour executer le serveur :
```
cd path_project/src
./serveur
```
